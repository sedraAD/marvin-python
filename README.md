# Marvin ‚Äì Python Menu Application

This project is a simple Python console application developed as part of an introductory programming course.

The application, called **Marvin**, presents a menu-based interface where the user can choose between different small features and exercises. The project was created to practice fundamental programming concepts in Python such as functions, loops, conditionals, input/output and basic problem-solving.

---

## Features

Marvin offers a menu with several options, including:

- Present yourself to Marvin
- Celsius to Fahrenheit conversion
- Points to grades conversion
- Calculate sum and average
- Repeat letters in a word
- Compare numbers (smaller, larger, equal)
- Validate and generate Swedish personal identity numbers
- Acronym generator
- Randomize letters
- Search country data
- Calculate emission changes

Each menu option represents a small independent exercise implemented in Python.
